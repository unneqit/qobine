## Web UI

The player can start an embedded web interface. This is disabled by default and must be started with the `--web` argument. It also listens on `0.0.0.0:9888` by default. Change port with `--port` argument.

Go to `http://localhost:9888` to view the UI.

## Installation
### Homebrew
`qobine-web` is available in [homebrew](https://formulae.brew.sh/formula/qobine-web)

### Download Release

Download the tar.gz file for your supported OS from the releases page, extract the file and execute `qobine-web` or copy it to your `$PATH`.
